참여 방법

  1. 1) 코드 받기

    GitHub 저장소 `https://github.com/jungcollin/series_game.git`의 `main` 최신 코드를 받아줘.
이미 저장소를 받은 폴더면 `git pull origin main` 하고,
아니면 원하는 작업 폴더에서 `git clone git@github.com:jungcollin/series_game.git` 한 뒤
저장소 루트로 이동해서 바로 다음 단계 진행해줘.

  2. 2) 만들기

    /make-stage [스테이지 설명 한 줄]

  3. 3) 확인하기

    /check-stage

  4. 4) 게임 올리기 요청

    /publish-stage
